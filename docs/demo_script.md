# 答辩演示脚本

## 演示顺序

1. 打开首页，展示系统首页、图谱接入状态、关系网络面板。
2. 展示普通问答：
   - 文彭是谁？
   - 文彭的字是什么？
   - 文徵明与文彭是什么关系？
3. 展示固定工具查询与对应 SPARQL。
4. 展示高级 SPARQL 页面。
5. 展示关系网络可视化面板。
6. 展示与 CBDB / CText 对齐后的扩展知识。

## 风险兜底

- 如果模型生成不稳定，优先展示固定工具查询。
- 如果图谱未完全对齐，优先展示核心图谱中的人物基础信息和关系。
- 如果高级查询结果为空，切回预设问句和固定 SPARQL 示例。
- 如果网络可视化边较少，说明当前查询结果尚未形成足够密集的关系子图。
